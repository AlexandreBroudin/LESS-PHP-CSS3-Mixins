# [LESS-PHP CSS3 Mixins starter kit](http://alexandrebroudin.net/article/less-css3,14.html)

Another LESS CSS3 Mixins starter kit, but with [LESS-PHP] (http://leafo.net/lessphp/)

## Features

* Border radius
* Box shadow
* Text shadow
* Background size
* Background rgba
* Linear gradients
* Radial gradients
* Transition
* Skew
* Scale
* Rotate
* Translate
* 3D transform
* Columns

## IE < 10

* NO IE FILTERS.
* -ms prefixes only
